Botjack

a restful bitcoin blackjack game

Welcome to BotJack!

BotJack is a RESTful API based bitcoin blackjack game built and designed for bots. Playing this game requires a curl wrapper and will not work in the web browser. Use the returned token field and then and pass it back in the Authorization header. Game play is completely stateless and the token automatically expires after 24 hours. The purpose of using a REST method for game play allows for continuous bot play. Check out the math and see if this can work out to your advantage.

Instructions:

  1. /create_player/[name]
  2. send bitcoin to the address returned and save the token
    NOTE: It may take up to 10 minutes for 2 confirmations to transfer bitcoin to the wallet.
  3. /balance - gets the current wallet balance, min of .01 BTC required to play
  4. /deal - shuffle and deal a new game
  5. /hit or /stand
  6. repeat and then /cashout/[wallet_address] to send the bitcoin to your wallet

Game Information:

  • Dealer stops at 17
  • No insurance, split, double down
  • ACE counts as 1 or 11
  • Current Bet is fixed at .01 BTC
  • Single Deck, a new shuffle each hand

Provability:

This game uses the php functions "srand" and "shuffle" to produce a deck. The deck is "shuffled" four times to ensure randomness. The code for shuffle is located here.

curl https://botjack.co/create_player/dan
